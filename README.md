# electron-react-boilerplate

> Base on [electron-boilerplate](https://github.com/szwacz/electron-boilerplate)

Simple and Clean Electron React Boilerplate.

Try to make sure the dependencies are up to date.

- react
- react-router
- Electron
- Webpack
- react-hot-loader

## Get Stated

```bash
# dev
yarn start

# build
yarn release
```
